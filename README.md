# Knucklebone AI

An AI capable of playing the game "Knucklebone" from Cult of the lamb

the functions are in the file create_game.py
they are :
- game_between_human
- game_between_robot
- game_with_human

they need a depth that the bot will look up to
for game_between_robot you can choose if they have the same depth by giving 2 different depth in parameter
